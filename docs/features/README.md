# Feature Docs

This directory is **transitional**.

Today it contains a mixed set of feature reference docs, design-heavy feature writeups, and example material. Long term it should shrink.

## Use it for now

- stable feature-specific reference material that does not yet have a better home

## Prefer these destinations when possible

- **`docs/design/`** for design-heavy or architecture-heavy feature docs
- **`docs/reference/`** for stable feature reference
- **`docs/plans/`** for active initiative planning

## Goal

Over time this directory should keep only true feature reference docs, with design-heavy and planning-heavy content moved elsewhere.
